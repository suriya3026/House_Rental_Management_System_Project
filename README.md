# House Rental Management System

A web-based application developed to simplify and manage house rental operations efficiently. This system helps property owners list available rental houses and allows tenants to search and request bookings based on their preferences.

## 📌 Features

- User Authentication System
- Property Listing Management
- Search Houses by:
  - Location
  - Price
  - Availability
- Booking Request Module
- Responsive User Interface
- Secure Data Storage using MySQL

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Php

### Database
- MySQL

## 🎯 Project Objective

The main objective of this project is to create an easy-to-use rental management platform that connects property owners and tenants in a simple and efficient way.

## 💡 Key Functionalities

- Owners can add, update, and manage rental properties
- Tenants can view available houses and send booking requests
- Responsive design for better user experience across devices
- Efficient data storage and retrieval using MySQL database
